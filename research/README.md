# Research & Analysis Files

This directory contains research, analysis, and planning documents related to the AutoHotkey v2 fine-tuning project.

## Contents

### Example Analysis & Guides
- **AHK_V2_EXAMPLES_COMPLETE_GUIDE.md** - Comprehensive guide to all AutoHotkey v2 examples
- **AHK_V2_ALPHA_MODULE_SYSTEM_GUIDE.md** - Guide to AHK v2 alpha module system
- **EXAMPLES_SUMMARY.md** - Summary of example files and categories
- **PRUNING_RECOMMENDATIONS.md** - Analysis and recommendations for removing redundant examples

### GitHub Repository Research
- **GITHUB_RESEARCH_REPORT.md** - Research findings from GitHub repositories
- **GITHUB_SCRAPING_GUIDE.md** - Guide for scraping GitHub for AHK v2 examples
- **JNIZM_REPOSITORY_REVIEW.md** - Review of jnizm's AutoHotkey repositories
- **LEXIKOS_LIBRARIES_REVIEW.md** - Review of Lexikos' AutoHotkey libraries
- **XYPHA_REPOSITORY_REVIEW.md** - Review of xypha's AutoHotkey repositories

### Local Analysis
- **LOCAL_DESKTOP_RESEARCH_PROMPT.md** - Prompts for local desktop research
- **LOCAL_LIBRARIES_COMPREHENSIVE_REVIEW.md** - Comprehensive review of local libraries
- **REPOSITORY_ANALYSIS_AHK_NOTES.md** - Notes from repository analysis
- **RESEARCH_PROMPT_AHK_V2_SCRIPTS.md** - Research prompts for AHK v2 scripts

## Purpose

These documents were created during the data collection and curation phase to:
1. Identify high-quality AutoHotkey v2 example sources
2. Analyze existing examples for quality and redundancy
3. Document the structure and organization of the training dataset
4. Plan improvements and cleanup operations

## Note

These are reference documents and should not be deleted. They provide valuable context for understanding the dataset's origins and curation decisions.
